PYTHAGOREAN THEOREM CALCULATOR

This application asks for two pieces of information regarding a right triangle to solve for the Pythagorean Theorem: one side length and one angle, or two side lengths. The Pythagorean Theorem is when both sides lengths are squared then added to each other should equal the square of the third length or the hypotenuse, as demonstrated in this format: a^2 + b^2 = c^2. Even if only one side of a right triangle is given alongside with one angle measure, then some trigonometry techniques must be applied first before using the Pythagorean Theorem to solve for the missing values. If one side and one angle are both given, then trigonometry techniques can also be used to solve for every missing value too. This application can be used to check your work when learning to master the Pythagorean Theorem and basic trigonometry techniques.

Technologies used: HTML, CSS, JavaScript
